BLOKD

A no-root Android ad and tracker blocker. One local VPN, and known ad domains stop resolving across every app.

Download APK Source

What it does

  • Runs a local VPN and answers DNS for ad and tracker domains with NXDOMAIN, so they never load.
  • Works system wide, across every app and browser. No root, no account.
  • Public block lists (HaGeZi and OISD), updatable in app, plus your own allow list.

Install

Download the latest BLOKD-vX.Y.Z.apk, open it, flip the switch, and approve the VPN prompt Android shows. The APK is release signed, so it installs and updates normally.

Honest limits

DNS blocking cannot remove ads stitched into a video stream from the same server as the content, which is what most large streaming apps use for in stream ads. It also cannot see encrypted DNS or hardcoded resolver IPs.
