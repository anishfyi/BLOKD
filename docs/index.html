BLOKD

A no-root Android ad and tracker blocker. One local VPN, and known ad domains stop resolving across every app.

Download APK Install guide

What it does

  • Runs a local DNS-only VPN and answers ad and tracker domains with NXDOMAIN.
  • Ships 560k+ bundled domains (HaGeZi + OISD), with daily CDN refresh in the background.
  • Optional encrypted AdGuard DNS-over-TLS adds a second filtering layer.
  • Dual-stack IPv4/IPv6 VPN config fixes false "no internet" warnings on modern Android.

Protection modes

Standard balanced

HaGeZi Pro Plus Mini + OISD. Strong coverage with lower memory use.

Berserk aggressive lists

Adds the full HaGeZi Pro list for maximum DNS-level blocking. Not full-tunnel interception — see limits.

Read more about modes →

Honest limits

DNS blocking cannot remove ads stitched into a video stream from the same server as the content. It also cannot see apps that use encrypted DNS (DoH/DoT) or hardcoded resolver IPs.
